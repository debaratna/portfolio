## 🚀 Featured Project

### [`angular-mfe-platform`](https://github.com/debaratna/angular-mfe-platform) — Production-Grade Microfrontend Shell

> A reference implementation of a Microfrontend platform using Angular 17+ and Webpack Module Federation, designed for distributed teams shipping independently.

```
🏗️  Shell host + 3 remote MFEs (Auth, Catalog, Dashboard)
⚡  Shared state via custom Angular Signals store (no NgRx boilerplate)
🔐  OIDC-aware routing with lazy-loaded remotes
📦  NX monorepo with build caching and affected-only CI
🌐  GDPR-compliant, OWASP-hardened, prod-ready
```
